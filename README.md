# ai-image

University project — image processing / AI experiment (Python)

Maintainer: Zhassulan Baimyshev — https://www.linkedin.com/in/zhassulan-baimyshev/

Summary

A small Python project exploring image processing and simple AI techniques. The main entrypoint is `main.py` and required Python packages are listed in `requirements.txt`.

Tech

- Language: Python
- Key files: `main.py`, `requirements.txt`

Quick start

1. Create a virtual environment and install dependencies:

```bash
python -m venv .venv
source .venv/bin/activate  # or .venv\Scripts\activate on Windows
pip install -r requirements.txt
```

2. Run the app:

```bash
python main.py
```
